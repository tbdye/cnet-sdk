# CNet SDK Headers

- `cnet/` -- CNet struct definitions and master include (`cnet.h`)
- `clib/` -- C prototypes for fd2pragma (input files for stub generation)
- `fd/` -- Function definition files (register assignments, LVO offsets)
- `inline/` -- GCC inline stubs generated by fd2pragma (LP-macro style)
- `pragma/` -- Original SAS/C pragma files (preserved for reference)
- `proto/` -- Proto headers that conditionally include inline/ or pragma/

Usage: compile with `-Iinclude -Iinclude/cnet` and `#include <cnet/cnet.h>`.
