# CNet Door Templates

Starter templates for new projects. Copy one as a starting point.

| Template | Use Case |
|----------|----------|
| `pfile_cnetc.c` | CNetC door (recommended for new development) |
| `pfile_raw.c` | Raw CMessage door (v3 compatible) |
| `standalone.c` | CLI utility accessing CNet data structures |

See [docs/getting-started.md](../docs/getting-started.md) for a walkthrough.
