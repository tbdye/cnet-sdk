# CNet SDK Tools

Diagnostic utilities for CNet development.

| Tool | Purpose |
|------|---------|
| `guoffset.c` | Print GETUSER field offsets for PortData, UserData, etc. |
| `libinfo.c` | Display CNet library versions and export counts |
| `structsizes.c` | Report sizeof() for all CNet structs |

Build with `make tools` from the project root.
